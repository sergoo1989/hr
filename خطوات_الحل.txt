═══════════════════════════════════════════════════════════
📋 الخطوات المطلوبة منك الآن
═══════════════════════════════════════════════════════════

┌─ الخطوة 1️⃣ : رفع API على Railway ─────────────────┐
│                                                         │
│  1. اذهب إلى: https://railway.app                     │
│  2. سجل دخول بـ GitHub                                 │
│  3. New Project → Deploy from GitHub repo             │
│  4. اختر repository الخاص بك                          │
│                                                         │
│  ⚙️ الإعدادات:                                         │
│     Root Directory: apps/api                          │
│     Build: npm install && npm run build               │
│     Start: npm run start:prod                         │
│                                                         │
│  🔧 Variables (مهم جداً!):                            │
│     NODE_ENV=production                               │
│     JWT_SECRET=نص-سري-طويل-32-حرف                    │
│     JWT_EXPIRES_IN=7d                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─ الخطوة 2️⃣ : احصل على رابط Railway API ──────────┐
│                                                         │
│  بعد deployment ستحصل على رابط مثل:                  │
│  https://hr-api-production-abc123.up.railway.app      │
│                                                         │
│  📝 احفظ هذا الرابط!                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─ الخطوة 3️⃣ : حدّث config.js ─────────────────────┐
│                                                         │
│  افتح الملف: frontend/config.js                       │
│  في السطر 13، غيّر:                                   │
│                                                         │
│  من:                                                   │
│  'https://your-app-name.up.railway.app'              │
│                                                         │
│  إلى:                                                  │
│  'https://hr-api-production-abc123.up.railway.app'   │
│  (ضع رابطك الحقيقي)                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘

┌─ الخطوة 4️⃣ : ارفع على Vercel مرة أخرى ───────────┐
│                                                         │
│  في terminal:                                          │
│  $ vercel --prod                                       │
│                                                         │
│  أو:                                                    │
│  $ git add .                                           │
│  $ git commit -m "Update API URL"                     │
│  $ git push                                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
