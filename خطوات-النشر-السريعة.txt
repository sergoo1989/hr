📋 الخطوة 1: رفع Backend (5 دقائق)
═══════════════════════════════════════════════════════════════

1. افتح المتصفح وروح على:
   👉 https://railway.app

2. اضغط "Login" → "Login with GitHub"
   (لو مش عندك حساب GitHub، اعمل واحد مجاني)

3. بعد تسجيل الدخول:
   ┌─────────────────────────────────────┐
   │ اضغط "New Project"                  │
   │ اضغط "Deploy from GitHub repo"      │
   │ اختار repository بتاع المشروع     │
   └─────────────────────────────────────┘

4. إعدادات Backend:
   في صفحة Settings:
   
   ┌─────────────────────────────────────┐
   │ Root Directory:                     │
   │ apps/api                            │
   │                                     │
   │ Build Command:                      │
   │ npm install && npm run build        │
   │                                     │
   │ Start Command:                      │
   │ npm run start:prod                  │
   └─────────────────────────────────────┘

5. Variables (متغيرات البيئة):
   اضغط على "Variables" وأضف:
   
   ┌─────────────────────────────────────┐
   │ NODE_ENV = production               │
   │ PORT = 3000                         │
   │ JWT_SECRET = احط-رقم-سري-123456     │
   │ JWT_EXPIRES_IN = 7d                 │
   └─────────────────────────────────────┘

6. اضغط "Deploy" واستنى 2-3 دقائق ⏳

7. لما يخلص، هتلاقي رابط زي:
   https://hr-system-production.up.railway.app
   
   📝 احفظ الرابط ده! مهم جداً ⬇️
   ______________________________________

✅ Backend جاهز! النظام بيشتغل دلوقتي على الإنترنت!

═══════════════════════════════════════════════════════════════

📋 الخطوة 2: تحديث Config (1 دقيقة)
═══════════════════════════════════════════════════════════════

1. ارجع للمشروع على جهازك

2. شغّل الملف:
   👉 إعداد-للنشر.bat

3. لما يطلب منك رابط API:
   الصق الرابط اللي حفظته من Railway
   مثال: https://hr-system-production.up.railway.app

4. اضغط Enter

✅ Config اتحدّث!

═══════════════════════════════════════════════════════════════

📋 الخطوة 3: رفع Frontend (5 دقائق)
═══════════════════════════════════════════════════════════════

1. افتح تاب جديد وروح على:
   👉 https://vercel.com

2. اضغط "Sign Up" → "Continue with GitHub"

3. بعد تسجيل الدخول:
   ┌─────────────────────────────────────┐
   │ اضغط "Add New..."                  │
   │ اضغط "Project"                     │
   │ اختار repository بتاع المشروع     │
   └─────────────────────────────────────┘

4. إعدادات Frontend:
   
   ┌─────────────────────────────────────┐
   │ Framework Preset: Other             │
   │ Root Directory: frontend            │
   │ Build Command: (سيبه فاضي)         │
   │ Output Directory: .                 │
   └─────────────────────────────────────┘

5. اضغط "Deploy" واستنى دقيقة ⏳

6. لما يخلص، هتلاقي رابط زي:
   https://hr-system.vercel.app
   
   📝 ده رابط النظام بتاعك! 🎉

✅ Frontend جاهز!
